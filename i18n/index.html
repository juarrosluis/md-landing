Control de validaciones y transacciones

Control de las operaciones más críticas del negocio como pueden ser las transacciones bancarias.

El procedimiento es que la empresa establezca un importe máximo por operación de manera que, cuando éste sea superado, se conectará con nuestro módulo de validación y notificará, vía APP, a los responsables para que validen la operación.

Esta validación dispone de reglas de negocio tales como: validación por biometría, control de horarios, control por geolocalización, lista negras de IP

Este Servicio puede ser consumido por cualquier aplicación de la empresa, al disponer de una API de acceso.
